Cupcake Sign

Cupcake cutout, can be used as a birthday sign.

Instructions

Edit this page




Just cut it out and paint it how you wish. We used chalkboard paint on part of the front so we can write the name of whoevers birthday it is. We cut a length of rope and stapled it to the back in order to hang it up.
